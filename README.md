# TEEGrow Foundation Website v5

Responsive static website for GitHub Pages.

## Stack
- Static HTML
- CSS
- Vanilla JS
- Poppins via Google Fonts
- Approved TEEGrow branding assets from the supplied brand board

## Deploy on GitHub Pages
1. Create a GitHub repository.
2. Upload all files in this folder to the repository root.
3. Go to Settings → Pages.
4. Select branch `main`, folder `/root`.
5. Add custom domain `teegrow.org`.
6. Update DNS as GitHub instructs.

## Before going live
Replace the placeholder Google Form link in `index.html`:

`https://forms.gle/`

with your actual Expression of Interest form link.
